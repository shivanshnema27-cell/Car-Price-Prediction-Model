# 🚗 Car Price Prediction App

This is an end-to-end Machine Learning project that predicts car prices based on various features.

## 🔧 Tech Stack
- Python
- Scikit-learn
- Streamlit
- NumPy

## 📊 Features
- Feature Engineering
- One-Hot Encoding
- Ridge/Lasso Regression Model
- Interactive Streamlit UI
